# DragForm Plugin

A plugin for creating forms using dead simple Drag&Drop

### Usage

```
<div id='dragform'></div>

<script src="path/to/dragform.min.js"></script>
<script>
  new DragForm();
</script>
```

### Development

```
git clone git@github.com:acolin/dragform.git
cd /path/to/repo
npm install

// Run the dev server at localhost:8080
webpack-dev-server
```

### Running tests

```
npm test
```
