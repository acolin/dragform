# DragForm Plugin

A plugin for creating forms using death simple Drag&Drop

### Usage

```
<div id='dragform'></div>

<script src="path/to/dragform"></script>
<script>
  $(function(){
    new DragForm();
  });
</script>
```

### Development

```
git clone
cd /path/to/repo
npm install

// Run the dev server at localhost:8080
webpack-dev-server
```
